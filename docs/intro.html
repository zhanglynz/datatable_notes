1  Introduction to data.table

https://cran.r-project.org/web/packages/data.table/vignettes/datatable-intro.html

if you are interested in reducing programming and compute time tremendously, then this package is for you

NB: Two type of efficiency: coder’s efficiency and code’s efficiency — reference: Colin Gillespie, C.; Lovelace, R. (2021). Efficient R programming. URL: https://csgillespie.github.io/efficientR/

File reader fread

  • data.table’s fast-and-friendly file reader fread
  • fread accepts http and https URLs directly
library(data.table)
link <- "https://raw.githubusercontent.com/Rdatatable/data.table/master/vignettes/flights14.csv"
flights <- fread(link)

1.1 Basics

  • data.table()
library(data.table)

set.seed(12345)

small_dtb <- data.table(x = letters[1:5],
                        y = rnorm(5),
                        z = 1:5)
small_dtb
        x          y     z
   <char>      <num> <int>
1:      a  0.5855288     1
2:      b  0.7094660     2
3:      c -0.1093033     3
4:      d -0.4534972     4
5:      e  0.6058875     5

You can also convert existing objects to a data.table using setDT() (for data.frame and list structures) or as.data.table() (for other structures).

  • General form DT[i, j, by] — Take DT, subset/reorder rows using i, then calculate j, grouped by by.

  • Subset rows in i

Example:

(small_dtb[x %in% c('a', 'b', 'c', 'e') & z >= 3])
        x          y     z
   <char>      <num> <int>
1:      c -0.1093033     3
2:      e  0.6058875     5

Example:

(small_dtb[c(2, 5)])
        x         y     z
   <char>     <num> <int>
1:      b 0.7094660     2
2:      e 0.6058875     5
  • Order rows

Example:

# x in in descending order; and then y in ascending order
(small_dtb[order(-x, y)])
        x          y     z
   <char>      <num> <int>
1:      e  0.6058875     5
2:      d -0.4534972     4
3:      c -0.1093033     3
4:      b  0.7094660     2
5:      a  0.5855288     1
  • Select column(s) in j

Example:

(small_dtb[, .(x, z)])
        x     z
   <char> <int>
1:      a     1
2:      b     2
3:      c     3
4:      d     4
5:      e     5

Tip: As long as j-expression returns a list, each element of the list will be converted to a column in the resulting data.table.

Example:

# select and rename
(small_dtb[, .(x_1 = x, z)])
      x_1     z
   <char> <int>
1:      a     1
2:      b     2
3:      c     3
4:      d     4
5:      e     5

Example:

# return a vector
(small_dtb[, z])
[1] 1 2 3 4 5
  • Compute or do in j

Example:

(small_dtb)
        x          y     z
   <char>      <num> <int>
1:      a  0.5855288     1
2:      b  0.7094660     2
3:      c -0.1093033     3
4:      d -0.4534972     4
5:      e  0.6058875     5
(chk <- small_dtb[, sum(y + z > 3)])
[1] 2
  • Subset in i and do in j

Example:

(small_dtb[x %in% letters[2:4], sum(y + z > 3)])
[1] 1

Example:

(small_dtb[x %in% letters[2:4], .(mean_1 = mean(y), mean_2 = mean(z))])
       mean_1 mean_2
        <num>  <num>
1: 0.04888851      3
  • Special symbol .N

.N is a special built-in variable that holds the number of observations in the current group.

Example:

(small_dtb[x %in% c('a', 'c', 'f') & y > 0, .N])
[1] 1
  • Select columns named in a variable using with = FALSE

Example:

sel_vars <- c('x', 'z')
(small_dtb[, sel_vars, with = FALSE])
        x     z
   <char> <int>
1:      a     1
2:      b     2
3:      c     3
4:      d     4
5:      e     5
  • Deselect columns using - or !

Example:

(small_dtb[, -c('x', 'z')])
            y
        <num>
1:  0.5855288
2:  0.7094660
3: -0.1093033
4: -0.4534972
5:  0.6058875
(small_dtb[, !c('x', 'z')])
            y
        <num>
1:  0.5855288
2:  0.7094660
3: -0.1093033
4: -0.4534972
5:  0.6058875
  • Select by specifying start and end column names (From version 1.9.5+)

Example:

(small_dtb[, y:z])
            y     z
        <num> <int>
1:  0.5855288     1
2:  0.7094660     2
3: -0.1093033     3
4: -0.4534972     4
5:  0.6058875     5

1.2 Aggregations

# create a data.table 
set.seed(98765)

dt <- data.table(ID = sprintf("%03d", 1L:100L))
dt[, `:=`(x = sample(letters[1:3],   length(ID), replace = TRUE),
          y = sample(month.abb, length(ID), replace = TRUE),
          w = runif(length(ID)),
          z = rnorm(length(ID))

  • Make ‘longer’ table — melt
fk_data <- data.table(week = 1:5,
                      player_1 = rnorm(5),
                      player_2 = rnorm(5))
long_tb <- melt(fk_data,
                measure.vars = c("player_1", "player_2"),
                variable.name = "player",
                value.name = "value")
(long_tb)
